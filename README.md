# Aoi Journal（青色 Journal）

青色申告のための複式簿記デスクトップアプリケーションです。  
UI ファーストの設計で、仕訳の入力からレポート生成まで一つのアプリで完結します。  
データはローカルのテキストファイル（Ledger 形式）に保存され、インターネット接続なしで動作します。

---

## できること

### 帳簿の管理
- **新規帳簿の作成** — 屋号・事業所名と事業年度を設定して帳簿を新規作成できます
- **帳簿ファイルの開閉・保存** — `.journal` 形式のテキストファイルを開いたり保存したりできます
- **期首残高の設定** — 前年度からの繰越残高を帳簿に記録できます

### 仕訳の入力
- **手動入力** — 日付・摘要・勘定科目・金額を直接入力して仕訳を登録できます
- **AI 入力（Ollama 連携）** — 自然言語（例:「3月1日 AWSの費用 12,345円 クレカ払い」）を入力すると、ローカル LLM が仕訳を自動提案します。提案内容は登録前に確認・修正できます
- **仕訳の編集・削除** — 登録済みの仕訳をあとから修正・削除できます
- **仕訳の検索** — 摘要や勘定科目名でフィルタリングできます
- **貸借バランスチェック** — 借方・貸方の合計が一致しない仕訳は登録できません

### 勘定科目
- **デフォルト科目プリセット** — 青色申告でよく使う勘定科目（現金、普通預金、売掛金、売上高、通信費、旅費交通費 など）があらかじめ用意されています
- **カスタム科目の追加** — 独自の勘定科目を追加できます（資産・負債・純資産・収益・費用の5種類）

### レポート
- **総勘定元帳（General Ledger）** — 科目ごとの取引明細と残高を確認できます
- **損益計算書（P/L）** — 期間内の収益・費用・当期純利益を確認できます
- **貸借対照表（B/S）** — 資産・負債・純資産の残高を確認できます

### その他
- **オフライン動作** — インターネット接続は不要です
- **プライバシーファースト** — データはすべてローカルに保存され、外部サーバーには送信されません

---

## できないこと

- **複数通貨の管理** — 日本円（JPY）のみに対応しています
- **消費税の自動計算・区分** — 課税・非課税・免税の区分や消費税額の自動計算には対応していません
- **確定申告書（青色申告決算書）の自動生成** — 仕訳データから確定申告書類を直接出力する機能はありません
- **CSV / Excel へのエクスポート** — レポートのエクスポートには対応していません
- **複数帳簿の同時管理** — 一度に開ける帳簿は1ファイルのみです
- **クラウド同期** — データの自動バックアップやデバイス間同期には対応していません
- **銀行口座・クレジットカードの自動インポート** — 明細の自動取得には対応していません
- **減価償却の自動計算** — 固定資産の減価償却スケジュール管理には対応していません
- **請求書・領収書の管理** — 書類の添付や管理機能はありません

## ファイルフォーマット

帳簿は Ledger ライクなプレーンテキスト形式（`.journal`）で保存されます。  
テキストエディタで直接閲覧・編集することも可能です。

```
; === Aoi Journal ===
; business: 山田太郎 フリーランス
; fiscal_year: 2024-01-01 - 2024-12-31

account 現金 type:asset
account 普通預金 type:asset
account 売上高 type:revenue
account 通信費 type:expense

2024-01-01 期首残高
    普通預金  500000 JPY
    元入金  -500000 JPY

; id:xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
2024-03-01 売上入金
    普通預金  100000 JPY
    売上高  -100000 JPY

; 領収書番号: 2024-0315-001
; id:xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
2024-03-15 AWS費用
    通信費  12345 JPY
    クレジットカード  -12345 JPY
```

## ビルド・開発手順

### 必要な環境

- [Rust](https://www.rust-lang.org/tools/install)（最新安定版）
- [Node.js](https://nodejs.org/)（v18 以上推奨）
- [Tauri CLI の前提条件](https://v2.tauri.app/start/prerequisites/)（OS ごとに異なります）

### セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run tauri dev

# リリースビルド
npm run tauri build
```

### テスト

```bash
# Rust バックエンドのテスト
cargo test --manifest-path src-tauri/Cargo.toml
```

## AI 入力（Ollama）の設定

自然言語から仕訳を自動提案する機能は [Ollama](https://ollama.com/) を使用します。  
Ollama を使わない場合は手動入力モードをご利用ください。

1. [Ollama をインストール](https://ollama.com/download)する
2. 使用するモデルを取得する（例: `ollama pull llama3.2`）
3. アプリの「設定」タブで Ollama URL・モデル名を設定する

デフォルト設定:
- URL: `http://localhost:11434`
- モデル: `llama3.2`
